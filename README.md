# KDT_SQL_mini_project

### 🔧 주제
**C++ TCP 소켓 기반 로그인 & 채팅 시스템 제작**

### 🎯 목표
1. C++로 TCP 소켓 통신을 구현  
2. MySQL 데이터베이스를 연동  
3. 회원가입, 로그인, 채팅 기능이 통합된 콘솔 프로그램 개발  

### 🧩 구현 기능 요약
- **회원가입**: 클라이언트가 ID/PW를 입력하여 서버에 등록 요청  
- **로그인**: 로그인 성공 시 채팅 메뉴로 진입 가능  
- **채팅**: 서버와 메시지 송수신 (Echo 방식)  
- **로그아웃**: 로그인 상태 초기화 후 초기 메뉴로 복귀  
- **종료**: 프로그램 완전 종료
